博客简介及搭建

Author ：Erics Lei

整个博客将包含 java 全栈开发知识、408考研四门课、Data Mining、数学等知识，杂而不精。

博客采用的框架是 hexo + nexT，图床部署在 github 并使用 jsDelivr 进行 cdn 加速。

共勉！

以下是博客美化过程，供参考。

主题美化

folk github 图标

  1. 去 https://tholman.com/github-corners/ 这里挑选自己喜欢的 github 图标样式

  2. 在博客源码下 next/layout/_layout.swig 找到 headband ，在下面添加复制的代码, href 改成自己的 github 地址

主页文件自定义

  1. 在 hexo 根目录的 source 目录的根目录下创建 index.md 文件

  2. 修改根目录下的 _config.yml 文件，将 index_generator 配置的 path 属性设置成无效值，我这里写的就是 useless

  3. next 目录下 config 文件的 menu 配置 home 属性保持默认即可

  4. 重新部署后主页文件即可修改成功
